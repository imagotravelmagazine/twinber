# Twinber - App di Compatibilità

Questa applicazione calcola la compatibilità tra due persone basandosi su un questionario dettagliato.

## Funzionalità
- Quiz interattivo con salvataggio dei progressi.
- Generazione di un codice univoco per ogni utente.
- Confronto dei codici per calcolare l'affinità.
- Supporto multilingua.
- Integrazione con Firebase per il salvataggio dei dati e chat in tempo reale.
